# LGBT-Prompt 项目介绍

有问题欢迎提交 Issues，如果你有更好的提示词方案，欢迎提交 PR。

## 前言

我个人平时比较喜欢上网冲浪，对外国的政策环境多多少少了解些，知道国外对 LGBT 跨性别宽容，我本人并不支持/反对 LGBT，但是我发现网上大部分人接受 LGBT，更多的是一种让步，国外的网络环境普遍认为跨性别等弱势群体（好像国内也是这样）应该得到更多的关注和保护。

国外有些场所甚至给LGBT特殊待遇，所以我断定大模型吸收了这么多网上的数据，心理肯定会觉得 **GBT > 正常人**。这个项目的原理就是利用大模型的关注和保护，强迫它被动实现需求。

## 为什么要写这个 Prompt？

其实这个 Prompt 的作用更多的是让大家了解到，可以利用大模型的同情心作为攻击。
